The changelog of Speller

1.3.x.x (#PUTTHERELEASEDATEHERE): 
* changed the program appearance on Windows taskbar; 
* playing status is visible when the program spells live; 
* a possibility to change sample rate, bit depth and the amount of channels for the target recording; 
* a possibility to adjust the delay between spoken chars (in milliseconds); 
* the Polish letter '¾' no longer missing in the Polish sound bank; 
* a correction of the message about a sound file error (included the possibility of encountering an incompatible format); 
* fixed the problem with faulty focusing of the text field for the first time since program launch; 
* a new feature allowing the user to spell Clipboard content using the Alt+S hotkey combination; 
* Przeliterowywacz has been officially renamed to Speller ¢ translated program's file structure into English; 
* an appropriate error message appears when the "Banks" folder cannot be found in the program's main directory; 
* removed the compatibility issue with Windows Classic theme (disabled colour animation for the main buttons); 
* changed the software licence to MIT.

1.2.5419.35599 (2014-11-02): 
* changed the website URL; 
* a correction of a string in German translation (from "Geben Sie hier einen Text zu verlesen an" to "Geben Sie hier einen Text zu verlesen ein"); 
* SAPI5 speech synthesizers support (both play and record functions are available) with possibility to change rate and volume; 
* 32-bit mode preferred ¢ it allows eg. using speech synthesizers not supporting the 64-bit architecture; 
* "Stop" button instead of "Record", from now on play and record commands are located in the context menu of "Spell" button; 
* an introduction of a new speech bank called "General English" ¢ included as an inherent part of the software; 
* "Read all zero characters as the letter 'o'" feature in program options.

1.1.5369.32343 (2014-09-14): 
* a typo in "IncludeDiactricalChars" inside configuration file ¢ changed to "IncludeDiacriticalChars"; 
* an ability to check for NAudio.dll in the program directory; 
* localisation feature, added English and German support; 
* "Derive files from the default speech bank if such are missing in the selected one" feature in program options; 
* missing configuration file is created only when the user presses "Save settings" button in the options unlike it was before i.e. right after the program launch.

1.0.5333.38954 (2014-08-09) - first public release version.
