The changelog of Przeliterowywacz

1.2 Beta 1: 
* changed the website URL, 
* correction of a string in German translation (from "Geben Sie hier einen Text zu verlesen an" to "Geben Sie hier einen Text zu verlesen ein"), 
* SAPI5 speech synthesizers support (both play and record functions are available), 
* 32-bit mode preferred – it allows eg. using speech synthesizers not supporting the 64-bit architecture.

1.1.5369.32343: 
* typo in "IncludeDiactricalChars" inside configuration file – changed to "IncludeDiacriticalChars", 
* ability to check for NAudio.dll in the program directory, 
* localisation feature, added English and German support, 
* "Derive files from the default speech bank if such are missing in the selected one" feature in program options, 
* missing configuration file is created only when the user presses "Save settings" button in the options unlike it was before i.e. right after the program launch.

1.0.5333.38954 - first public release version.
